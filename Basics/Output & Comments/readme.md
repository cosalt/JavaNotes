
## 💬 Comments Used
- `//` - Single-line comment
- `/* */` - Multi-line comment

## 📝 Notes
- `System.out.print()` - Prints without newline
- `System.out.println()` - Prints with newline
- `\n` - Manual newline character
- `sout + tab` - IntelliJ shortcut for `System.out.println()`
